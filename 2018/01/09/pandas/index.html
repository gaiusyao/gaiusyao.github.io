2.文件读取

  这里以读取csv文件为例，介绍3种常用的数据读取函数：

  • df = pd.read_csv(path=’file.csv’)
  • df = pd.read_json(‘file.json’) #可以传入json格式字符串
  • df = pd.read_excel(‘file.xls’, sheetname=[0,1..]) #读取多个sheet，返回多个df的字典
