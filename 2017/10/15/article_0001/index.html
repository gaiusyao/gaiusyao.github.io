Hexo常用命令

  1. hexo g(hexo generate) 生成静态文件
  2. hexo s(hexo server) 启动本地web服务，用于预览
  3. hexo d(hexo deploy) 部署到远端（如github）
  4. hexo n(hexo new) 新建页面或文章
  5. hexo v(hexo version) 查看版本信息
  6. hexo help 查看命令帮助
  7. hexo clean 清除缓存文件 (db.json) 和已生成的静态文件 (public)
  8. hexo list 列出网站资料
